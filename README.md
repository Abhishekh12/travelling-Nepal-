# 🇳🇵 Travelling Nepal — Feel the Nature

> A tourist guide website for visitors exploring Nepal, built as my first-year university project.  
> **Grade Awarded: 99% — University of East London (2022)**

---

## 🌄 About The Project

**Travelling Nepal** is a full tourist guide website designed to help international visitors discover the beauty of Nepal — featuring destinations, accommodation options, travel tips, and a help & contact section for travellers who need assistance.

The project was built entirely from scratch using core web technologies, with a focus on clean design, real content, and a smooth user experience. A live video background of Pokhara was integrated into the homepage to give visitors an immersive first impression of Nepal.
